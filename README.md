# Discord RPG

## Installation
```
npm install
```

**Production**
```
npm run build
npm run start
```
**Development**
```
npm run watch
npm run dev
```

## config.yml
```
discord:
  token: <bot_token>
default:
  ressources:
    wood: 0
    stone: 0
    gold_nugget: 0
    money: 0
```

## Commands:
* [ ] !help - commands list
* [ ] !mine - harvest resources
* [ ] !ressources - ressources list
* [ ] !money - display money
