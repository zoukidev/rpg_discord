# RPGDiscord

## Installation
```
npm install
```

**Production**
```
npm run build
npm run start
```
**Development**
```
npm run watch
npm run dev
```

## config.yml
```
discord:
  token: <bot_token>
```

## Commands:
* [x] !setchannel - set channel for using bot (need Administrator role)
* [x] !help - commands list
* [x] !ressources - ressources list
* [x] !money - display money
* [ ] !mine - harvest resources
