# Discord RPG

## Installation
```
npm install
```

**Production**
```
npm run build
npm run start
```
**Development**
```
npm run watch
npm run dev
```

## config.yml
```
discord:
  token: <bot_token>
```

## Commands:
* [ ] !help - commands list
* [ ] !mine - harvest resources
* [ ] !ressources - ressources list
* [ ] !money - display money

## Utils
[Discord](https://discord.com/), [Discord.js](https://discord.js.org/)
